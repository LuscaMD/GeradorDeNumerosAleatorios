## 📝 Funcionalidades: 

- Gerar quantos numeros o usuário desejar;
- Sortear entre numeros negativos e positivos (ex: -10 e 25);
- Sortear entre numeros positivos (ex: 1 e 8);
- Separar os numeros sorteados por vírgulas para facilitar a leitura.

<br>

## 🚀 Tecnologias

- HTML  
- CSS
- Java Script
